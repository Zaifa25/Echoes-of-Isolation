# Echoes of Isolation

## Overview

Echoes of Isolation is a voice-responsive paranormal horror game developed as a Final Year Project using Unreal Engine 5. The game delivers an immersive psychological horror experience where the player’s real-world voice directly affects gameplay and enemy behavior.

Set inside an abandoned psychiatric hospital, players must explore dangerous environments, solve ritual-based puzzles, collect hidden relics, and survive terrifying paranormal entities while remaining silent to avoid detection.
